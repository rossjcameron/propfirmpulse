# PropFirmPulse

PropFirmPulse is a mobile-first trading companion app built for prop firm futures traders.

It helps traders track account rules, log trades quickly, monitor progress towards funded status, and review journal entries with screenshots and detailed trade notes.

## Features

- Track prop firm account balances, drawdown limits, and profit targets
- Log trades with:
  - strategy
  - direction
  - outcome
  - entry and exit times
  - confidence level
  - emotional state
  - notes and lessons
  - optional screenshots
- View a journal organised by trading day
- Open detailed trade entries with full information
- Monitor progress towards funded status
- View account health and rule-aware account metrics
- Premium mobile UI built with Expo Router and React Native

## Tech Stack

- React Native
- Expo
- Expo Router
- TypeScript

## Purpose

PropFirmPulse is designed to answer three key questions for prop firm traders:

1. Can I take this trade safely?
2. How far am I from funded?
3. Am I following my rules?
